## 핵심 포인트 요약

### 배경
- MCP/A2A 생태계 확장 → AgentHub 실험 → 제품화 전환 욕구

### 비전
- 플랫폼 종속적 콘텐츠 소비에서 **사용자 주권적 통합 인터페이스**로의 전환

### 구현 방향
- **Local-First 아키텍처** (중앙 서버 없음, 사용자 디바이스에서 전부 처리)
- 콘텐츠 유형별 규격화 → LLM 기반 수집·변환 → 사용자 맞춤 UI 렌더링

### 권고 MVP
- **유튜브 동영상 + 관련 뉴스/이슈 통합 표시 데스크톱 앱** (단일 플랫폼부터 시작)
- 로컬 실행, 사용자 본인 API Key, 오픈소스 배포
